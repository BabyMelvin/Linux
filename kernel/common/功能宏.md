# 2.typecheck

```c
#define typecheck(type, x) \
({ type __dummy; \
   typeof(x) __dummy2; \
   (void)(&__dummy == &__dummy2); \
	1;\
})
```

* 后面就是取两个变量的地址，然后比较，因为__dummy是double类型而__dummy2是int类型，取地址后是两种不同类型的指针，**不同类型的指针不能比较**，会出现**编译错误**，这种方式实现了类型的检查。
* 这个是表达式最后的值。如果整个表达式没有错误，那么最后的1就是整个表达式的值。
