# 位运算
## 1.信息打印

```c
int a = 0;
printf("%0x\n",a);//16进制
printf("%0X\n",a);//16进制
printf("%d\n",a); //10进制
printf("%o\n",a); //8进制
```

16进制将4位二进制为一位16进制：如 0x25-->0010 0101

8进制将3位二进制为一位8进制：如 o73----->111 101
## 2.位运算符号

位运算操作数必须是整数类型，遵循寻常算术转换.

* `&`:位与运算:特定位清零。位掩码使用.`a&=0xFF;`高位清零,`~0x20`比`0xFFFFFFDF`更受欢迎,可以更好移植性，也方便阅读。
* `|`:位或运算:设定某位
* `^`:位异或运算
* `~`:位NOT运算
* `>>`:位右移运算
* `<<`:位左移运算
