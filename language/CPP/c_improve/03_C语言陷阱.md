# 1.连接
一个C程序由多个分别编译的部分组成，不同部分通过连接器的程序合并成一个整体。编译器一般每次只处理一个文件，**不能检测出那些需要一个了解多个源程序才能察觉的错误**。某些程序提供lint程序，如果有需要多加利用。

## 1.1 头文件
每个外部对象只在一个地方声明。这个声明地方一般就在一个头文件中，需要用到该外部对象的所有模块都应该包含这个头文件。需要特别指出，定义该外部对象的模块也应该包括这个头文件
